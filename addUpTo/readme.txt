Kolla på videon för att se hur sidan ska fungera.

Se video_grid_fill för att se hur den delen ska fungera (samma för alla sidor).
Säkerställ att allting fungerar som det ska efter det att griden fylls på nytt.

OBS:
Det måste handla om två olika celler. De kan dock ha samma siffra i sig.
Om det inte finns två celler som adderar till siffran användaren har skrivit så händer ingenting på sidan.
